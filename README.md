GIT: É uma ferramenta para criar e gerenciar versoes de codigos no projeto atual, para asegurar que o código logo após ser modificado e vir a quebrar após adicionar uma funcao,
é possivel acessar o backup salvo e retornar aonde funcione, isso ajuda bastante em trabalho individuais quanto em grupos.

GITHUB: Funciona como uma "Rede Social" para desenvolvedores/programadores, aonde permite que os dev's armazenem os seus protejotos dentro de REPOSITORIOS.
Os repositorios eles podem ser tanto publicos como privados, e esses repositorios ficam salvos em um Armazenamento em Nuvem vinculado a Conta.
